Voici la version professionnelle en anglais optimisée pour [Google AI Studio](https://aistudio.google.com?utm_source=chatgpt.com) et les IA de génération de code.
Elle précise bien que le frontend doit être uniquement en Angular et non en React.

Develop a modern university sports competition management web application for IUSJ (Institut Universitaire Saint-Jean).

IMPORTANT:

* The frontend MUST be developed ONLY with Angular 21 (or Angular 17 minimum).
* DO NOT use React.
* DO NOT use Next.js.
* Use Angular standalone architecture.
* Use TypeScript everywhere.
* Backend API must use Django REST Framework.

========================
PROJECT OVERVIEW
================

Create a modern SaaS-style university sports management platform used to organize:

* Interclass matches
* Inter-department competitions
* Tournaments
* University sports events

The platform must centralize:

* Teams management
* Players management
* Match scheduling
* Scores
* Rankings
* Notifications
* Sports administration

The platform is dedicated to IUSJ schools:

* Saint Jean Ingénieur
* Saint Jean School of Management
* Prépa Vogt
* CPGE
* Prépa Saint Jean Douala

Each team must belong to one school.

========================
AUTHENTICATION & ROLES
======================

Implement JWT Authentication with:

* Login
* Register
* Logout
* Token storage
* HTTP Interceptor
* AuthGuard
* Role-based access control

Available roles:

* Admin
* Sports Manager
* Team Captain
* Student

IMPORTANT ROLE RULES:

* Every new user registering on the platform must automatically receive the "Student" role by default.
* Only the Admin can change user roles.
* Admin accounts must NOT be created from Angular frontend.
* Admin users must be created manually from Django Admin Panel only.
* Role management must be secured.
